This is MaintainHub, an Enterprise CMMS platform built with Clean Architecture. Tech stack: pnpm monorepo with Turborepo, Node.js 20, TypeScript strict mode, Fastify v4, Prisma + PostgreSQL, Redis, BullMQ, Anthropic SDK, Next.js 15 App Router, Tailwind CSS, shadcn/ui. Follow SOLID principles. Every domain entity must have unit tests. Use conventional commits. Never hardcode secrets. All mutations must write to audit log.
